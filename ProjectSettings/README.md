# ProjectSettings

This folder contains Unity project configuration files.

**Important:** These files are required for Unity to recognize this as a valid project.

## Key Files

- `ProjectVersion.txt` - Specifies the Unity version (2022.3.25f1)
- `ProjectSettings.asset` - Main project configuration (generated by Unity)
- `EditorSettings.asset` - Editor preferences (generated by Unity)
- `InputManager.asset` - Input mappings (generated by Unity)
- `TagManager.asset` - Tags and layers (generated by Unity)

## Note

Most files in this folder are auto-generated by Unity and excluded from version control via `.gitignore`. Only `ProjectVersion.txt` is committed to ensure version consistency.
