# Getting Started - Robot Tower Defense

## ✅ You're All Set!

Everything has been installed and configured. Here's what to do next:

---

## 🎮 **Step 1: Launch the Game**

### Option A: Automatic (Recommended)
**Double-click:** `Launch Game.bat`

This will:
- Check if Unity is installed
- Install Unity if needed
- Open the project in Unity Editor

### Option B: Manual
1. Open Unity Hub
2. Click "Open" and select this folder
3. Wait for Unity to import assets (5-10 minutes first time)
4. Double-click `MainMenu` scene in Project window
5. Click Play (▶) button

---

## 🎯 **Step 2: Start Playing**

Once Unity loads:

1. **Find the MainMenu scene**
   - Look in the Project window at the bottom
   - Navigate to: `Assets/Scenes/MainMenu`
   - Double-click to open it

2. **Press Play**
   - Click the ▶ (Play) button at the top center
   - The game will start in the Game tab

3. **Follow the Tutorial**
   - On first launch, the tutorial will auto-start
   - It teaches you the basics in 9 easy steps
   - You can skip it from the settings if desired

---

## 📚 **Quick Tips**

1. **Start with Campaign Mode**
   - Begin with the first map
   - Complete 30 waves to unlock the next map

2. **Understand Tower Types**
   - **Gun Tower**: Balanced, good starting tower
   - **Cannon**: High damage, slow fire rate
   - **Laser**: Fast fire rate, low damage
   - **Missile**: Area damage, good for groups
   - **Sniper**: Long range, high single-target damage

3. **Manage Your Resources**
   - Starting credits: 1,000
   - Earn credits by defeating enemies
   - Balance offense and economy

4. **Upgrade Strategically**
   - Each tower has 3 upgrade tiers
   - Upgrade key towers first
   - Don't spread resources too thin

5. **Use Targeting Modes**
   - **First**: Enemies closest to exit
   - **Last**: Enemies furthest from exit
   - **Strongest**: Highest health
   - **Weakest**: Lowest health
   - **Closest**: Nearest to tower

## 🏆 **Progression System**

### Achievements (65 Total)
- Complete objectives to unlock achievements
- Bronze, Silver, and Gold tiers
- View progress in the Achievements menu

### Tech Tree (15 Upgrades)
- Permanent upgrades that persist between games
- Earn Tech Points by completing waves
- Unlock new abilities and boosts

### Mission System
- **Daily Missions**: 3 new missions every 24 hours
- **Weekly Missions**: 3 new missions every 7 days
- Complete missions for bonus rewards

## ❓ **Need Help?**

### Common Issues

**Unity won't start?**
- Run `Setup-Game.ps1` again
- Check Task Manager for Unity processes
- Restart your computer

**Assets not loading?**
- Wait for Unity to finish importing (check bottom-right progress bar)
- If stuck, close Unity and delete `Library` folder, then reopen

**Game runs slowly?**
- Open Settings → Graphics
- Select "Low" quality preset
- Reduce resolution

**Can't build for Android?**
- Install Android Build Support via Unity Hub
- Go to: Installs → [Your Unity Version] → Add Modules
