# Game Icon Setup

## Current Status
A desktop shortcut has been created: **Robot Tower Defense.lnk**

## To Use Your Custom Icon:

### Option 1: For the Desktop Shortcut
1. Right-click the "Robot Tower Defense" shortcut on your desktop
2. Select "Properties"
3. Click "Change Icon..."
4. Click "Browse..."
5. Navigate to: `C:\Users\tydaw\OneDrive\Documents\RobotTowerDefense\Assets\Icon\`
6. Select your icon image (you'll need to save the AI image here as .ico format)
7. Click OK

### Option 2: For Unity Build Icon
When building your game:
1. Save your icon image in this folder as a .png file (multiple resolutions recommended)
2. In Unity: Edit → Project Settings → Player → Icon
3. Assign your icon images for different platforms

### Converting the AI Image to Icon Format:
Since Windows shortcuts require .ico format, you can:
1. Save the AI-generated image from your message as a PNG
2. Use an online converter (like https://convertio.co/png-ico/)
3. Save the .ico file in this Assets/Icon folder
4. Update the shortcut properties as described above

## Icon Specifications:
- **Windows Desktop**: .ico format (multiple sizes: 16x16, 32x32, 48x48, 256x256)
- **Unity Editor**: Any image format (PNG recommended)
- **iOS**: Various sizes (1024x1024 for App Store)
- **Android**: Adaptive icon (foreground + background layers)
